# zotero-csl-studio
Personal collection of modified Zotero CSL citation styles.

本仓库中的 CSL 文件主要用于个人学习、写作和研究场景。样式可能不完全符合期刊、学校或出版社的最新要求。正式投稿或提交论文前，请以目标机构、期刊或出版社的官方格式要求为准。

# 许可证

除非单个文件另有说明，本仓库中的修改版 CSL 样式遵循原始 CSL 样式的许可证要求。若样式来源于 CSL 官方样式库，请保留原作者和贡献者信息，并遵循 Creative Commons Attribution-ShareAlike 3.0 Unported License。
